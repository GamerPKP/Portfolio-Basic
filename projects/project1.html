AI Essay Assistant

This project is a simple web app that helps students generate essay outlines using AI. It was built using HTML, CSS, and a lightweight JavaScript API call to OpenAI’s model. The focus was on clean UI and user-friendly layout.

HTML CSS JavaScript

Features

  • Generates essay ideas and structure instantly.
  • Responsive layout for all devices.
  • Uses async JS functions for smooth interaction.

What I Learned

I learned how to manage API keys securely, structure HTML for clarity, and apply CSS transitions effectively. I also improved my ability to debug asynchronous JavaScript calls.
